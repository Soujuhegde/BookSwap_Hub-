
## 🐛 Common Issues & Solutions

### Issue: Page shows blank/white screen
**Solution:** Check browser console for errors, usually missing imports

### Issue: Protected pages redirect to login
**Solution:** This is expected! Login first at `/login`

### Issue: Icons not showing
**Solution:** Check if RemixIcon CDN is loaded in `index.html`:
```html
<link href="https://cdn.jsdelivr.net/npm/remixicon@4.0.0/fonts/remixicon.css" rel="stylesheet">
```

### Issue: API calls failing
**Solution:** Ensure backend is running on http://localhost:3000
